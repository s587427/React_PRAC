實用
    container
        Breakpoint
            sm (640px)	 max-width: 640px;
            md (768px)	 max-width: 768px;
            lg (1024px)	 max-width: 1024px;
            xl (1280px)	 max-width: 1280px;
            2xl (1536px) max-width: 1536px;
    布局
        flex
        shrink-{num}
        p-{num]
        max-w-{size}
        mx-auto => center it horizontally
    外觀
        w-{num} => 寬,  h-{num} => 高
        bg-{color}-{deep} => background color
        rounded-{size} => border radius
        shadow-{size} => box-shadow   
        space-between => innerItem space
    字體
        text-{size} => font size
        text-{color} => text color
        font-{type} => font-weight

    like that required, invalid, and disabled
